## Move ssh-keys
```
sudo curl https://github.com/equa-tory.keys >> ~/.ssh/authorized_keys

sudo sed -i 's/^#\?PasswordAuthentication .*/PasswordAuthentication yes/' /etc/ssh/sshd_config
sudo systemctl restart sshd
```
